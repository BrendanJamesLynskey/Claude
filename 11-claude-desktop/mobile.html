Claude Desktop

The Native Claude App

One window, three tabs — Chat, Code, and Cowork — plus a global hotkey, tray icon, connectors, and native notifications.

Native App Chat Code Cowork
01

What Is It?

Claude Desktop is the native app for Claude on macOS and Windows. A signed-in wrapper around claude.ai with system integration a browser can’t give you.

Chat

  • Conversation, Projects, Artifacts
  • Files, images, voice
  • Extended thinking, Styles

Cowork

  • Autonomous background agent
  • Runs in an Anthropic-hosted VM
  • Dispatch from your phone

Code

  • Interactive pair programming
  • Local files, visual diffs
  • Same engine as the CLI

Why native? Global hotkey, menu-bar/tray icon, drag-and-drop, native notifications, and installable MCP connectors.

Cowork Tab

Hand Claude a task and walk away. Each task runs in an Anthropic-hosted VM with its own filesystem, network, and tools.

Typical Uses

  • Research and comparison writeups
  • Clone repo, run tests, open PR
  • Monitor a PR queue
  • Draft status emails from tickets

Free in the VM

  • Browser, terminal, Python, Node, Git
  • Persistent workspace
  • Live video of progress
  • Dispatch from phone

Caveat

macOS Cowork requires Apple Silicon (M1+). Windows: all supported hardware.

  • Tool-call in/out

  • Anonymous telemetry — opt out

Enterprise

  • Connector and model allow-lists
  • Audit log of chats and tool calls
  • Bedrock / Vertex routes

MDM

  • Silent install via Jamf / Intune
  • Lock shortcut, disable Cowork, pin region
10

MCP Transports

Three transports: stdio (local subprocess), Streamable HTTP (current remote standard), and deprecated HTTP+SSE.

stdio

  • Subprocess on your machine, stdin/stdout
  • Works on: CLI, Chat tab, Code tab
  • Cannot reach the network

Streamable HTTP

  • Single HTTP(S) endpoint, POST + optional SSE
  • Current standard (March 2025 spec)
  • CLI, Chat, Code → plain HTTP + private IPs OK
  • Cowork, Web, Mobile, API → HTTPS + CA cert required

Key Distinction

CLI / Chat tab / Code tab connect directly from your machine. Cowork / Web / Mobile / API are brokered by Anthropic and require public HTTPS.

Private Network

CLI, Chat, Code connect from your machine — plain HTTP and LAN IPs work: http://10.60.12.34:3001/mcp. Brokered surfaces need public HTTPS — tunnel via ngrok if needed.

Config Gotcha

Chat tab → claude_desktop_config.json. Code tab → ~/.claude.json. Separate — servers don’t cross.

12

Troubleshooting

Sign-in Loops

  • Quit fully, relaunch, sign in again
  • Clear OAuth from the keychain
  • 403 → sign out from app menu

Cowork Missing

  • macOS: check uname -m → arm64
  • Free plans: no Cowork tab
  • Region rollout — see Account

Connectors

  • Local ones need the target app installed
  • Remote ones need OAuth
  • Corporate firewall may block MCP WebSocket

Logs

  • ~/Library/Logs/Claude (mac)
  • %APPDATA%\Claude\logs (win)
  • Help → Send Feedback attaches them
